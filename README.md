# Mandesh Security Suite v1.2.2 🇰🇪

A comprehensive Python-based security auditing and educational toolkit designed for **Mandesh Online Academy**. This suite provides tools for network auditing, penetration testing simulations, and mobile security research.

## 🚀 Features

* **Payload Generator:** Multi-platform generator (Android Ghost APKs, Windows EXEs, Linux Shells).
* **Intelligence Hub:** OSINT tools for subdomain discovery and leak hunting.
* **Network Auditor:** Integrated Metasploit handler with persistent Serveo tunneling.
* **Forensic Mechanic:** Metadata (EXIF) extraction and image analysis tools.
* **Persistence Lab:** Educational scripts for understanding crontab and boot-start persistence.

## 🛠️ Installation

1. Clone the repository:
   bash
   git clone [https://github.com/YOUR_USERNAME/mandesh-security-suite.git](https://github.com/YOUR_USERNAME/mandesh-security-suite.git)
   cd mandesh-security-suite
2. Install dependencies:

Bash
pip install -r requirements.txt
sudo apt install apktool default-jdk docker.io -y
3. Run the main interface:

Bash
python3 main.py
